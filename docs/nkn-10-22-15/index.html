Normalized Metadata

Linked, Normalized Metadata (HCLS)

HCLS - Health Care and Life Sciences W3C

"a guide to indicate what are the essential metadata, and the manner in which we can express it"

Unfortunately HCLS does not explicitly include geographical information. But since it is only meant to encapsulate the essentials, there is no restriction on adding additional fields. Linked data allows us to transparently add new fields as long as we define them or link to existing definitions...
